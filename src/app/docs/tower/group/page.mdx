# Group Acceptance

This module provides you the functionality of accepting or declining join requests for your Roblox groups
through RoWifi.

## Setup

To setup group acceptance, follow the setup guide [here](/docs/tower/setup).

## Commands

### Accepting a Join Request

#### **/group accept**

#### **Parameters**
- **`group`**: The ID of the Group.
- **`username`**: The user to accept in the group.

### Declining a Join Request

#### **/group decline**

#### **Parameters**
- **`group`**: The ID of the Group.
- **`username`**: The user to decline from joining the group.
