# XP Binds

XP Binds are a type of bind that allow you to award XP to your users and promote them in the Roblox group when they meet the requirements.

A XP Bind consists of four elements:

- **Group**: The ID of your Roblox Group
- **Rank**: The ID of the rank in the Roblox Group. This will range from 1-255.
- **XP**: The XP at which to promote the user to this rank.

### Availability

XP Binds are available for Gamma Tier servers. You can create as many XP Binds as you want.

### Creating XP Binds

XP Binds can be created on the [dashboard](https://dashboard.rowifi.xyz) or using the command `/xpbinds new`.

### Modifying XP Binds

XP Binds can be modified individually on the [dashboard](https://dashboard.rowifi.xyz).

### Deleting XP Binds

XP Binds can be deleted using the [dashboard](https://dashboard.rowifi.xyz) or by the command `/xpbinds delete`.
