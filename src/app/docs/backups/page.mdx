# Backups

Backups is a premium module that lets you backup your server configuration and import it into any servers you own. They are tied to your Discord User.

## Creating a Backup

To create a backup, run the command `/backup new`. It requires only one argument: `name`.

## Importing a Backup

To import an existing backup into a server, run the command `/backup restore`. It requires only one argument: `name`. This is
the name you provided when you created the backup.

## Viewing Saved Backups

To view a list of saved backups, run the command `/backup view`.
